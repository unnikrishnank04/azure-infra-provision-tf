# This architecture diagram represents a microservices-based web application deployed on Azure Kubernetes Service (AKS). It's designed to be highly available, secure, scalable, reliable, and cost-effective


Frontend in Angular hosted on AKS.

Backend with microservices 

Elasticsearch for product search.

Cosmos DB for user data.

SQL Database for Data store

Prometheus for monitoring.

Azure DevOps for CI/CD with Helm for deployment.
